# 🌴 East Belitung Tourism Landing Page

A responsive tourism landing page developed to promote the tourism potential of East Belitung Regency. This project was created during my internship at the Department of Communication, Informatics, Statistics, and Encryption (Diskominfo) of East Belitung Regency.

## 📖 About the Project

This landing page provides information about tourist destinations in East Belitung in an attractive and user-friendly interface. It is designed to improve accessibility to tourism information while delivering a responsive experience across desktop and mobile devices.

## ✨ Features

- Responsive landing page
- Modern and clean user interface
- Tourism destination information
- News and activity section
- Interactive navigation
- Mobile-friendly design

## 🛠️ Built With

- PHP
- HTML5
- CSS3
- JavaScript

## 🚀 Getting Started

1. Clone this repository

```bash
git clone https://github.com/anandakhairu/Lawang_Beltim.git
```

2. Move the project to your local server directory (e.g. XAMPP `htdocs`).

3. Start Apache using XAMPP.

4. Open your browser and visit:

```
http://localhost/Lawang_Beltim
```
